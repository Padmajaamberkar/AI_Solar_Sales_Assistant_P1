# ☀️ AI Solar Sales Assistant

An AI-powered Solar Sales Assistant that helps customers understand solar solutions, estimate their requirements, and get quick responses using Artificial Intelligence.

## 📌 Project Overview

The **AI Solar Sales Assistant** is designed to simplify the solar sales process by providing an interactive AI chatbot experience. It helps users with solar-related queries, basic system recommendations, and customer support.

This project combines **Artificial Intelligence, Python, and Data Science concepts** to create a smart assistant for the solar energy industry.

## 🎯 Problem Statement

Many customers interested in solar energy have questions regarding:

* Solar system requirements
* Suitable capacity estimation
* Benefits of solar installation
* Cost-related queries
* General solar information

Traditional sales processes require manual support. This AI assistant provides faster and automated responses.

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

### 2. Open the project folder

```bash
cd AI_Solar_Sales_Assistant
```

### 3. Install required libraries

```bash
pip install -r requirements.txt
```

### 4. Add your API Key

Create an environment variable for your Groq API key.

Example:

```
GROQ_API_KEY = "your_api_key_here"
```

### 5. Run the application

```bash
streamlit run app.py
```

The application will open in your browser.

## 📊 Solar Calculation Logic

The assistant includes solar capacity estimation based on user requirements.

Example:

* 0–1100 units → 1 kW Solar System
* 1100–2200 units → 2 kW Solar System

*(The calculation logic can be extended for larger requirements.)*
